# 快速入门

本章节将帮助您在 5 分钟内开始使用 ComAssistant。

## 第一步：启动程序

双击桌面上的 ComAssistant 图标，或从开始菜单中启动程序。

## 第二步：连接串口

1. **选择串口**: 在工具栏的"选择串口"下拉框中选择您的串口设备（如 COM3）
2. **设置波特率**: 选择合适的波特率（常用 115200）
3. **打开连接**: 点击"打开串口"按钮

连接成功后，状态栏会显示"已连接: COMx"。

## 第三步：发送数据

### 文本模式
1. 在底部发送区输入文本
2. 点击"发送"按钮或按 Enter 键

### HEX 模式
1. 勾选"HEX发送"选项
2. 输入十六进制数据（如 `AA BB CC DD`）
3. 点击"发送"按钮

## 第四步：接收数据

接收到的数据会自动显示在接收区：

- **文本标签页**: 显示 ASCII 文本
- **十六进制标签页**: 显示 HEX 格式
- **过滤标签页**: 根据关键词过滤显示

### 接收选项
- **时间戳**: 显示接收时间
- **自动滚动**: 自动滚动到最新数据
- **HEX显示**: 切换十六进制显示

## 第五步：保存数据

### 导出数据
1. 点击菜单 **文件 → 导出数据**
2. 选择导出格式（TXT、CSV、HTML、JSON）
3. 选择保存位置

### 保存会话
1. 点击菜单 **文件 → 保存会话**
2. 选择保存位置
3. 下次可通过 **文件 → 加载会话** 恢复

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+O | 打开/关闭串口 |
| Ctrl+L | 清空接收区 |
| Ctrl+E | 导出数据 |
| F1 | 打开帮助 |
