# 交互式控件

ComAssistant 提供强大的交互式控件系统，用于实时调参和数据可视化。

## 概述

交互式控件面板允许您：
- 实时调节设备参数
- 显示姿态数据（IMU/飞控）
- 自动构建并发送数据帧
- 保存和加载控件配置

## 打开控件面板

1. 菜单 **工具 → 控件面板**
2. 或使用快捷键

## 支持的控件类型

### 滑动条控件

用于实时调节数值参数。

**功能特点**：
- 拖动滑动条实时发送数据
- 可直接编辑名称（点击名称区域）
- 自定义数值范围和步进
- 自定义发送模板
- 发送频率限制（防止过于频繁）

**右键配置选项**：
- **发送模板**：定义发送数据格式
  - `%s` - 控件名称
  - `%.Nf` - N位小数的数值（如 `%.2f` 表示2位小数）
  - 示例：`SET %s %.2f\n` 发送 `SET Param1 50.00\n`
- **最小值/最大值**：数值范围
- **步进**：每次调节的最小单位
- **小数位数**：显示精度
- **单位**：显示单位（如 mV, Hz, %）
- **发送间隔**：拖动时最小发送间隔（毫秒）

**使用示例**：
```
配置：
  名称: KP
  发送模板: PID_KP=%.3f\r\n
  范围: 0.0 - 10.0
  步进: 0.001

拖动滑动条到 5.123 时发送:
  PID_KP=5.123\r\n
```

### 3D姿态显示控件

用于实时显示 IMU 或飞控的姿态数据。

**功能特点**：
- 3D 飞行器模型实时旋转显示
- 支持 Roll（横滚）、Pitch（俯仰）、Yaw（偏航）三轴
- 鼠标交互：拖动旋转视角，滚轮缩放
- 显示坐标轴和参考网格

**视图操作**：
- **左键拖动**：旋转视角
- **滚轮**：缩放

**数据通道绑定**：
- 横滚通道：接收 Roll 数据（-180° ~ +180°）
- 俯仰通道：接收 Pitch 数据（-90° ~ +90°）
- 偏航通道：接收 Yaw 数据（0° ~ 360°）

**右键配置选项**：
- **发送模板**：可选的发送格式
  - `%r` - 横滚角
  - `%p` - 俯仰角
  - `%y` - 偏航角
- **通道绑定**：选择数据通道索引
- **显示坐标轴**：是否显示 XYZ 坐标轴
- **显示网格**：是否显示参考网格

**嵌入式代码示例**：
```c
// 发送姿态数据到 ComAssistant
void sendAttitude(float roll, float pitch, float yaw) {
    // 使用 TEXT 协议
    printf("{plot1}%.2f,%.2f,%.2f\n", roll, pitch, yaw);

    // 或使用 JustFloat 协议
    float data[3] = {roll, pitch, yaw};
    sendJustFloat(data, 3);
}
```

## 控件管理

### 添加控件

1. 在控件类型下拉框中选择控件类型
2. 点击"添加"按钮
3. 新控件将出现在面板中

### 删除控件

- **单个删除**：右键点击控件 → 选择"删除"
- **清空所有**：点击"清空"按钮

### 配置控件

右键点击任意控件，选择"配置..."打开配置对话框。

### 保存/加载配置

- **保存**：点击"保存"按钮，选择保存位置（.json 格式）
- **加载**：点击"加载"按钮，选择配置文件

配置文件包含所有控件的类型、名称、参数设置等信息。

## 数据发送

当您操作控件时（如拖动滑动条），控件会：
1. 根据发送模板构建数据帧
2. 通过当前串口/网络连接发送
3. 发送间隔受限制，避免数据拥堵

**注意**：确保串口/网络已连接，否则数据不会发送。

## 数据绑定

3D姿态控件支持绑定数据通道，当绘图协议解析出数据时：
1. 根据通道索引获取对应的数值
2. 自动更新控件显示

**通道索引说明**：
- 通道从 0 开始计数
- 例如：TEXT 协议 `{plot1}1.2,3.4,5.6` 中，通道0=1.2, 通道1=3.4, 通道2=5.6

## 应用场景

### PID 参数调节

```
配置三个滑动条：
- KP (0-10, 步进0.01)
- KI (0-1, 步进0.001)
- KD (0-5, 步进0.01)

发送模板: SET_PID %s %.3f\n
```

### 飞控姿态监控

```
配置一个 3D 姿态控件：
- 绑定通道 0 到横滚
- 绑定通道 1 到俯仰
- 绑定通道 2 到偏航

设备端发送: {plot1}roll,pitch,yaw
```

### 电机速度控制

```
配置滑动条：
- 名称: Motor1
- 范围: 0-100 (占空比 %)
- 发送模板: M1=%d\r\n
```

## 常见问题

**Q: 拖动滑动条没有发送数据？**
A: 检查串口是否已连接，以及发送模板是否正确配置。

**Q: 3D姿态显示不更新？**
A: 确认已正确绑定数据通道，且设备正在发送对应格式的数据。

**Q: 控件配置丢失？**
A: 使用"保存"功能将配置导出为 JSON 文件，下次使用时"加载"即可恢复。
