# 数据表格视图

数据表格视图以结构化表格形式展示串口收发的数据，便于数据分析和追踪。

## 概述

数据表格视图提供：
- 实时显示收发数据
- 多列信息展示（时间戳、方向、HEX、ASCII等）
- 数据过滤和搜索
- CSV导出功能

## 打开数据表格

菜单 **工具 → 数据表格**

## 表格列说明

| 列名 | 说明 |
|------|------|
| 序号 | 数据帧序号，从1开始递增 |
| 时间戳 | 数据收发的精确时间 |
| 方向 | TX（发送）或 RX（接收） |
| HEX | 十六进制格式数据 |
| ASCII | ASCII文本格式数据 |
| 解析值 | 协议解析出的数值（如有） |
| 协议 | 使用的协议类型 |
| 描述 | 自动生成的数据描述 |

## 工具栏功能

| 按钮 | 功能 |
|------|------|
| 过滤框 | 输入关键字实时过滤数据 |
| 方向筛选 | 选择显示 全部/TX/RX |
| 暂停 | 暂停实时更新 |
| 清空 | 清除所有记录 |
| 导出 | 导出为CSV文件 |
| 复制 | 复制选中行到剪贴板 |

## 数据过滤

### 关键字过滤

在过滤框中输入文本，表格会实时显示匹配的记录：
- 匹配HEX列内容
- 匹配ASCII列内容
- 匹配描述列内容

### 方向筛选

使用下拉框选择：
- **全部**：显示所有数据
- **TX**：仅显示发送的数据
- **RX**：仅显示接收的数据

## 数据导出

### 导出为CSV

1. 点击"导出"按钮
2. 选择保存位置和文件名
3. CSV文件包含所有列数据

**CSV格式示例**：
```csv
序号,时间戳,方向,HEX,ASCII,解析值,协议,描述
1,2026-01-21 10:30:15.123,RX,48 65 6C 6C 6F,Hello,,TEXT,
2,2026-01-21 10:30:15.456,TX,4F 4B 0D 0A,OK,,ASCII,
```

### 复制选中行

1. 选中一行或多行（按住Ctrl多选）
2. 点击"复制"按钮或按Ctrl+C
3. 数据复制到剪贴板，可粘贴到其他程序

## 自动滚动

- **启用**：勾选"自动滚动"，新数据到来时自动滚动到底部
- **禁用**：取消勾选，可手动浏览历史数据

## 暂停/继续

- 点击"暂停"按钮停止实时更新
- 再次点击恢复更新
- 暂停期间的数据会被缓存，恢复后显示

## 数据限制

- 默认最大保留 10000 条记录
- 超过限制时，最早的记录会被自动删除
- 可在设置中调整此限制

## 使用场景

### 协议调试

查看每帧数据的详细内容，分析通信协议是否正确。

### 数据追踪

通过时间戳追踪数据收发的时序关系。

### 问题排查

使用过滤功能快速定位异常数据。

### 数据记录

导出CSV用于离线分析或报告。

## 常见问题

**Q: 数据更新很慢？**
A: 数据表格采用批量更新机制（100ms间隔），这是正常的性能优化。

**Q: 记录数量有限制吗？**
A: 默认限制10000条，可在设置中调整。过多记录可能影响性能。

**Q: 如何只看某个方向的数据？**
A: 使用工具栏的方向筛选下拉框，选择TX或RX。

**Q: 导出的CSV乱码？**
A: CSV文件使用UTF-8编码，请用支持UTF-8的软件打开（如Excel需要选择正确编码）。
