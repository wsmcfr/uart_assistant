## 系统要求

- **操作系统**: Windows 10 或更高版本
- **处理器**: 1 GHz 或更快
- **内存**: 512 MB RAM
- **硬盘空间**: 100 MB 可用空间
- **显示器**: 1280 x 720 或更高分辨率

## 主要功能

### 通信功能
- 串口通信（支持高波特率）
- TCP 客户端/服务器
- UDP 通信

### 数据处理
- 实时数据收发
- HEX/ASCII 格式转换
- 时间戳显示
- 数据高亮和过滤
- 智能滚屏（向上滚动自动暂停）
- 数据分窗（按规则路由数据）
- 数据表格视图

### 绘图功能
- 实时曲线绘图
- 多窗口支持与同步
- TEXT/STAMP/CSV/JustFloat 协议
- 直方图视图
- XY 绘图模式（李萨如图形）
- 数据导出

### 交互式控件
- 控件面板系统
- 滑动条调参控件
- 3D姿态显示控件（IMU/飞控姿态）
- 实时参数调节
- 控件配置保存/加载

### 数据分析
- 数据表格视图（实时数据监控）
- 数据过滤与搜索
- CSV数据导出

### 高级功能
- Lua 脚本支持
- 快捷发送
- 会话保存/恢复
- 多语言支持

## 快速开始

1. 启动 ComAssistant
2. 在工具栏选择串口和波特率
3. 点击"打开串口"按钮
4. 在发送区输入数据并发送
5. 在接收区查看收到的数据

详细使用说明请参阅各章节文档。
