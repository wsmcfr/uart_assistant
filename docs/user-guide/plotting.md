# 绘图功能

ComAssistant 提供强大的实时数据绘图功能，支持多种数据协议。

## 概述

绘图功能可以将串口接收的数据实时显示为曲线图，适用于：
- 传感器数据监控
- 波形分析
- 数据趋势观察

## 创建绘图窗口

1. 点击菜单 **绘图 → 新建绘图窗口**
2. 或使用工具栏上的"协议"下拉框选择绘图协议

## 支持的协议

### TEXT 协议

**格式**: `{窗口ID}数据1,数据2,数据3,...`

**示例**:
```
{plot1}1.23,4.56,7.89
{plot1}2.34,5.67,8.90
```

**说明**:
- 窗口ID：指定数据显示在哪个绘图窗口
- 数据：逗号分隔的数值，每个数值对应一条曲线
- X轴：自动递增

### STAMP 协议

**格式**: `<时间戳>{窗口ID}数据1,数据2,...`

**示例**:
```
<1000>{plot1}1.23,4.56
<2000>{plot1}2.34,5.67
```

**说明**:
- 时间戳：毫秒级时间戳，用作X轴
- 支持非均匀采样

### CSV 协议

**格式**: `窗口ID,X值,Y1,Y2,Y3,...`

**示例**:
```
plot1,0.1,1.23,4.56,7.89
plot1,0.2,2.34,5.67,8.90
```

**说明**:
- 第一列：窗口ID
- 第二列：X轴值
- 后续列：Y轴值（多曲线）

### JustFloat 协议（VOFA+ 兼容）

**格式**: 二进制数据，每个浮点数4字节（小端序IEEE754），以 `00 00 80 7F` 结尾

**帧结构**:
```
[float1][float2]...[floatN][00][00][80][7F]
```

**示例**（发送 1.0, 2.0, 3.0）:
```
HEX: 00 00 80 3F 00 00 00 40 00 00 40 40 00 00 80 7F
```

**说明**:
- 高效二进制协议，适合高速数据传输
- 每个浮点数占4字节，小端序
- 帧尾固定为 `00 00 80 7F`（对应浮点数 NaN）
- 与 VOFA+ 软件协议兼容

**嵌入式代码示例**:
```c
// STM32 发送 JustFloat 数据
void sendJustFloat(float* data, int count) {
    const uint8_t tail[] = {0x00, 0x00, 0x80, 0x7F};
    for (int i = 0; i < count; i++) {
        HAL_UART_Transmit(&huart1, (uint8_t*)&data[i], 4, 100);
    }
    HAL_UART_Transmit(&huart1, tail, 4, 100);
}
```

## 视图模式

### 时间序列模式（默认）

标准波形显示，X轴为时间/序号，Y轴为数据值。

### 直方图模式

将曲线数据以直方图形式显示，用于分析数据分布。

**启用方法**:
1. 菜单 **视图 → 直方图**
2. 选择数据源曲线和分箱数量

**功能特点**:
- 自动计算数据分布
- 显示均值、标准差统计信息
- 可调整分箱数量（10-200）

### XY 绘图模式

将两个通道的数据作为 X-Y 坐标绘制，适用于：
- 李萨如图形显示
- 相位分析
- 传感器校准曲线

**启用方法**:
1. 菜单 **视图 → XY模式** 或快捷键 `Ctrl+Y`
2. 选择 X 通道和 Y 通道

**示例应用**:
- 发送正弦和余弦信号，显示圆形
- 发送两个相关变量，观察相关性

## 多窗口同步

当有多个绘图窗口时，可启用同步功能：

**启用方法**: 菜单 **绘图 → 窗口同步**

**同步内容**:
- **暂停/继续状态**：一个窗口暂停，所有窗口同步暂停
- **X轴范围**：拖动缩放时，其他窗口同步调整X轴
- **清空操作**：清空一个窗口时，可选择同步清空所有窗口

**使用场景**:
- 对比多个通道的数据变化
- 同时观察多个传感器的数据
- 分析不同数据源的时间相关性

## 绘图窗口操作

### 工具栏功能

| 按钮 | 功能 |
|------|------|
| 暂停/继续 | 暂停或继续数据更新 |
| 清空 | 清除所有曲线数据 |
| 自动缩放 | 自动调整坐标轴范围 |
| 网格 | 显示/隐藏网格 |
| 图例 | 显示/隐藏图例 |
| 保存图片 | 导出为 PNG/JPG/PDF |
| 导出数据 | 导出为 CSV 文件 |

### 交互操作

- **缩放**: 滚动鼠标滚轮
- **平移**: 按住鼠标右键拖动
- **框选放大**: 按住鼠标左键拖动

## 性能优化

对于高速数据流：

1. **限制数据点数**: 默认保留最近 10000 个数据点
2. **降低更新频率**: 每 50ms 刷新一次显示
3. **启用 OpenGL 加速**: 在设置中开启（如果支持）

## 嵌入式设备代码示例

### Arduino 示例

```cpp
void loop() {
    float temperature = readTemperature();
    float humidity = readHumidity();

    // TEXT 协议
    Serial.print("{plot1}");
    Serial.print(temperature);
    Serial.print(",");
    Serial.println(humidity);

    delay(100);
}
```

### STM32 示例

```c
void sendPlotData(float temp, float hum) {
    printf("{plot1}%.2f,%.2f\r\n", temp, hum);
}
```

## 常见问题

**Q: 曲线不显示？**
A: 检查协议格式是否正确，确保选择了正确的绘图协议。

**Q: 数据更新卡顿？**
A: 尝试减少数据发送频率，或启用 OpenGL 加速。

**Q: 如何显示多个窗口？**
A: 使用不同的窗口ID，如 `{plot1}` 和 `{plot2}`。
