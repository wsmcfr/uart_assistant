# 更新日志

本文档记录 ComAssistant 的版本更新历史。

## [1.2.0] - 2026-01-21

### 新增功能

#### 绘图增强
- **直方图视图**：支持将曲线数据以直方图形式显示，自动计算分布统计
- **XY 绘图模式**：支持选择两个通道作为 X-Y 坐标绘制（李萨如图形等）
- **多窗口同步**：多个绘图窗口可同步暂停/继续状态
- **JustFloat 协议**：支持 VOFA+ 风格的二进制浮点协议，传输效率更高

#### 交互式控件（VOFA+ 风格）
- **控件面板**：可添加多种交互式控件进行实时调参
- **滑动条控件**：拖动即可发送参数，支持自定义发送模板
- **3D姿态显示控件**：实时显示 IMU/飞控的 Roll/Pitch/Yaw 姿态数据
- **控件单独删除**：右键菜单支持删除单个控件
- **控件配置保存**：支持保存/加载控件面板配置

#### 数据管理
- **智能滚屏**：向上滚动时自动暂停刷新，滚动到底部自动恢复
- **数据分窗**：根据关键字/正则表达式将数据路由到不同窗口显示
- **数据表格视图**：以表格形式查看收发数据，支持过滤、搜索、导出

### 改进
- 绘图窗口视图模式切换优化，直方图/XY模式数据持续更新时正常显示
- 绘图协议选择增加 JustFloat 选项

### 技术特性
- 新增 JustFloat 协议解析器（二进制 IEEE754 浮点数组 + 帧尾 00 00 80 7F）
- PlotterManager 增加窗口同步机制

## [1.0.0] - 2026-01-16 (正式发布)

### 新增功能

#### 通信模块
- 串口通信支持（自动检测、高波特率 115200-921600）
- TCP 客户端/服务器通信（底层支持）
- UDP 通信支持（底层支持）

#### 工作模式
- **串口模式**：标准串口收发，支持文本/HEX显示
- **终端模式**：类终端交互，支持 Ctrl+C/D/Z 控制字符
- **帧模式**：自定义帧格式（帧头/帧尾/校验）
- **调试模式**：数据记录、断点、时间差分析

#### 数据显示
- 多标签数据显示（文本、十六进制、过滤）
- 时间戳显示
- 数据统计（发送/接收字节数）
- 快捷发送功能（10组预设）

#### 曲线绘图
- 实时曲线绘图（QCustomPlot）
- TEXT 协议支持：`{windowId}data1,data2,...`
- STAMP 协议支持：`<timestamp>{windowId}data1,...`
- CSV 协议支持：`windowId,x,y1,y2,...`
- 频谱分析（FFT）
- OpenGL 加速绘图

#### 脚本与工具
- Lua 脚本引擎
- 工具箱（校验和、进制转换、编码转换）
- CRC16/CRC32 计算
- Base64/URL 编码转换
- IEEE754 浮点数转换

#### 数据安全
- 会话保存/恢复（.cas 格式）
- 数据导出（TXT、CSV、HTML、JSON）
- 自动保存功能（30秒间隔）
- 崩溃恢复机制

#### 界面与国际化
- 主题切换（明亮/暗黑）
- 多语言支持（中文/英文）
- 帮助文档系统
- 高 DPI 支持

### 技术特性
- 基于 Qt 6.8.x 开发
- C++17 标准
- 模块化架构设计
- 单元测试覆盖（44个测试用例）
- NSIS 安装包支持

### 已知限制
- 仅支持 Windows 10/11 (x64)
- 最大绘图点数：10000点
- 最大接收缓冲：1MB
- TCP/UDP 功能需通过工具栏切换（v1.1.0 新增）

## 版本号说明

版本号格式：`主版本.次版本.修订号`

- **主版本**：重大功能更新或不兼容的 API 变更
- **次版本**：新功能添加，向后兼容
- **修订号**：Bug 修复和小改进
