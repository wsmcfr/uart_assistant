示例1：温度监测 / Example 1: Temperature Monitoring

场景描述 / Scenario

中文：使用Arduino读取DHT22温湿度传感器，每秒通过串口发送数据到PC进行实时绘图。

English: Use Arduino to read DHT22 temperature/humidity sensor, send data via serial port to PC every second for real-time plotting.

Arduino代码 / Arduino Code

#include <DHT.h>

#define DHTPIN 2
#define DHTTYPE DHT22
DHT dht(DHTPIN, DHTTYPE);

void setup() {
    Serial.begin(115200);
    dht.begin();
}

void loop() {
    float temp = dht.readTemperature();
    float humidity = dht.readHumidity();

    // 使用TEXT协议发送
    Serial.print("{temp}");
    Serial.print(temp);
    Serial.print(",");
    Serial.println(humidity);

    delay(1000);  // 每秒一次
}

串口助手设置 / ComAssistant Settings

  • 波特率 / Baud Rate: 115200
  • 协议 / Protocol: TEXT绘图

发送的数据示例 / Data Example

{temp}25.3,60.2
{temp}25.4,60.1
{temp}25.5,60.3
{temp}25.6,60.2

预期结果 / Expected Result

  • 自动创建名为 "temp" 的绘图窗口 / Auto-create plot window named "temp"
  • 曲线1（蓝色）：温度数据 / Curve 1 (blue): Temperature data
  • 曲线2（红色）：湿度数据 / Curve 2 (red): Humidity data
  • X轴自动递增（0, 1, 2, 3...）/ X-axis auto increments

示例2：三轴加速度计 / Example 2: 3-Axis Accelerometer

场景描述 / Scenario

中文：使用MPU6050加速度计，实时监测X、Y、Z三轴加速度。

English: Use MPU6050 accelerometer to monitor X, Y, Z axis acceleration in real-time.

Arduino代码 / Arduino Code

#include <Wire.h>
#include <MPU6050.h>

MPU6050 mpu;

void setup() {
    Serial.begin(115200);
    Wire.begin();
    mpu.initialize();
}

void loop() {
    int16_t ax, ay, az;
    mpu.getAcceleration(&ax, &ay, &az);

    // 转换为g值
    float gx = ax / 16384.0;
    float gy = ay / 16384.0;
    float gz = az / 16384.0;

    // 使用STAMP协议（包含时间戳）
    Serial.print("<");
    Serial.print(millis());
    Serial.print(">{accel}");
    Serial.print(gx, 3);
    Serial.print(",");
    Serial.print(gy, 3);
    Serial.print(",");
    Serial.println(gz, 3);

    delay(50);  // 20Hz采样率
}

串口助手设置 / ComAssistant Settings

  • 波特率 / Baud Rate: 115200
  • 协议 / Protocol: STAMP绘图

发送的数据示例 / Data Example

<1050>{accel}0.125,0.458,0.991
<1100>{accel}0.124,0.457,0.990
<1150>{accel}0.126,0.459,0.992

预期结果 / Expected Result

  • 窗口名称：accel
  • 曲线1：X轴加速度 / Curve 1: X-axis acceleration
  • 曲线2：Y轴加速度 / Curve 2: Y-axis acceleration
  • 曲线3：Z轴加速度 / Curve 3: Z-axis acceleration
  • X轴为时间戳（毫秒）/ X-axis is timestamp (ms)

示例3：Modbus设备读取 / Example 3: Modbus Device Reading

场景描述 / Scenario

中文：通过Modbus RTU协议读取电表的电压、电流、功率数据。

English: Read voltage, current, power data from power meter via Modbus RTU protocol.

发送命令 / Send Command

中文：读取地址为01的设备，从寄存器0x0000开始读取3个保持寄存器

English: Read 3 holding registers from address 0x0000 of device 01

01 03 00 00 00 03 05 CB

串口助手设置 / ComAssistant Settings

  • 波特率 / Baud Rate: 9600
  • 数据位 / Data Bits: 8
  • 停止位 / Stop Bits: 1
  • 校验位 / Parity: None
  • 协议 / Protocol: Modbus
  • HEX发送 / HEX Send: 启用 / Enabled

响应数据示例 / Response Data Example

01 03 06 00 E6 00 3C 04 B0 XX XX

解析 / Parsing:

  • 01 - 设备地址 / Device Address
  • 03 - 功能码（读保持寄存器）/ Function Code (Read Holding Registers)
  • 06 - 字节数 / Byte Count
  • 00 E6 - 寄存器1值（230V）/ Register 1 value (230V)
  • 00 3C - 寄存器2值（60A）/ Register 2 value (60A)
  • 04 B0 - 寄存器3值（1200W）/ Register 3 value (1200W)
  • XX XX - CRC16校验 / CRC16 Checksum
💡 提示 / Tip: 接收区会自动解析Modbus帧并显示字段含义
Receive area will auto-parse Modbus frames and display field meanings

示例4：CSV格式频率响应 / Example 4: CSV Format Frequency Response

场景描述 / Scenario

中文：使用信号发生器和示波器测量滤波器的频率响应曲线。

English: Measure filter frequency response curve using signal generator and oscilloscope.

Python测试脚本 / Python Test Script

import serial
import time

ser = serial.Serial('COM3', 115200)

frequencies = [10, 20, 50, 100, 200, 500, 1000, 2000, 5000, 10000]

for freq in frequencies:
    # 设置信号发生器频率
    set_frequency(freq)
    time.sleep(0.5)

    # 读取示波器幅度
    amplitude = read_amplitude()

    # 发送CSV格式数据
    data = f"freq_resp,{freq},{amplitude}\n"
    ser.write(data.encode())

    time.sleep(0.5)

ser.close()

示例5：多通道数据采集 / Example 5: Multi-Channel Data Acquisition

场景描述 / Scenario

中文：使用ADS1115 ADC同时采集4路模拟信号。

English: Use ADS1115 ADC to simultaneously acquire 4 analog signals.

Arduino代码 / Arduino Code

#include <Wire.h>
#include <Adafruit_ADS1X15.h>

Adafruit_ADS1115 ads;

void setup() {
    Serial.begin(115200);
    ads.begin();
    ads.setGain(GAIN_ONE);
}

void loop() {
    // 读取4路ADC
    int16_t adc0 = ads.readADC_SingleEnded(0);
    int16_t adc1 = ads.readADC_SingleEnded(1);
    int16_t adc2 = ads.readADC_SingleEnded(2);
    int16_t adc3 = ads.readADC_SingleEnded(3);

    // 转换为电压
    float v0 = adc0 * 0.1875 / 1000.0;
    float v1 = adc1 * 0.1875 / 1000.0;
    float v2 = adc2 * 0.1875 / 1000.0;
    float v3 = adc3 * 0.1875 / 1000.0;

    // TEXT协议发送
    Serial.print("{adc}");
    Serial.print(v0, 3);
    Serial.print(",");
    Serial.print(v1, 3);
    Serial.print(",");
    Serial.print(v2, 3);
    Serial.print(",");
    Serial.println(v3, 3);

    delay(100);  // 10Hz采样率
}

预期结果 / Expected Result

  • 窗口名称：adc
  • 4条曲线分别代表CH0、CH1、CH2、CH3 / 4 curves for CH0, CH1, CH2, CH3
  • 实时显示各通道电压变化 / Real-time display of voltage changes on each channel
