快速入门 Quick Start

1. 串口连接 / Serial Connection

中文：

  1. 在工具栏选择串口端口（如 COM3）
  2. 选择波特率（默认 115200）
  3. 点击"打开串口"按钮

English:

  1. Select serial port in toolbar (e.g., COM3)
  2. Select baud rate (default 115200)
  3. Click "Open Port" button
💡 提示 / Tip: 连接前请确保端口未被其他程序占用。
Make sure the port is not occupied by other programs before connecting.

2. 发送数据 / Send Data

中文：在发送区输入数据，点击"发送"按钮

English: Enter data in send area and click "Send" button

  • 文本模式 / Text Mode: 直接输入文本 / Enter text directly
  • HEX模式 / HEX Mode: 输入十六进制（如 AA BB CC）/ Enter hexadecimal (e.g., AA BB CC)

示例 / Examples:

文本发送 / Text Send:

Hello World

HEX发送 / HEX Send:

AA 55 01 02 03 04

3. 接收数据 / Receive Data

中文：数据会自动显示在接收区的不同标签页

English: Data will be automatically displayed in different tabs of receive area

  • 文本标签 / Text Tab: 显示文本格式 / Display text format
  • 十六进制标签 / Hex Tab: 显示HEX格式 / Display HEX format
  • 过滤标签 / Filter Tab: 根据关键词过滤 / Filter by keywords

4. 常用功能 / Common Features

时间戳 / Timestamp

勾选"时间戳"选项，每行数据前会显示时间。
Check "Timestamp" option to show time before each line.

自动滚动 / Auto Scroll

勾选"自动滚动"，接收区会自动滚动到最新数据。
Check "Auto Scroll" to automatically scroll to latest data.

HEX显示 / HEX Display

勾选"HEX显示"，数据以十六进制格式显示。
Check "HEX Display" to show data in hexadecimal format.

循环发送 / Auto Send

勾选"循环发送"并设置间隔（毫秒），数据会定时自动发送。
Check "Auto Send" and set interval (ms) to send data automatically.

快捷发送 / Quick Send

中文：界面左侧有快捷发送区域，可预设常用指令，一键点击发送。

English: Quick send panel on the left side allows preset commands with one-click sending.

  • 点击 + 按钮添加新指令 / Click + button to add new command
  • 右键点击按钮可编辑、删除或调整顺序 / Right-click button to edit, delete or reorder
  • 支持文本和HEX格式 / Supports both text and HEX format
  • 可选择是否追加换行符 / Option to append newline
💡 提示 / Tip: 快捷发送配置会自动保存，下次启动时恢复。
Quick send configuration is automatically saved and restored on next launch.

关键词高亮 / Keyword Highlight

中文：在接收区可以设置关键词高亮，方便快速识别重要信息。

English: Set keyword highlighting in receive area to quickly identify important information.

  • 勾选"高亮"启用功能 / Check "Highlight" to enable
  • 点击"设置"按钮配置高亮规则 / Click "Settings" to configure rules
  • 可自定义前景色和背景色 / Customize foreground and background colors
  • 默认高亮 OK (绿色)、ERROR/FAIL (红色) / Default highlights: OK (green), ERROR/FAIL (red)

5. 快捷键 / Shortcuts

快捷键 / Key 功能 / Function
Ctrl+N 新建会话 / New Session
Ctrl+S 保存会话 / Save Session
Ctrl+O 加载会话 / Load Session
Ctrl+L 清空全部 / Clear All
F1 帮助文档 / Help

6. 版本更新 / Version Update

检查更新 / Check for Updates

中文：可通过菜单 帮助 -> 检查更新... 手动检查新版本。
English: Use menu Help -> Check for Updates... to manually check new releases.

自动检查 / Auto Check

中文：可在 帮助 -> 启动时自动检查更新 开启或关闭自动检查（默认每日一次）。
English: Toggle auto-check in Help -> Auto Check Updates on Startup (once per day by default).
