高级功能 Advanced Features

1. 宏录制/回放 Macro Recording

中文：宏功能允许您录制一系列发送操作，并在之后重复播放。

English: Macro function allows you to record a series of send operations and replay them later.

录制宏 / Recording

  1. 打开 工具 → 宏录制/回放 / Open Tools → Macro Recording
  2. 点击"开始录制"按钮 / Click "Start Recording"
  3. 在主界面进行正常的发送操作 / Perform normal send operations
  4. 点击"停止录制"保存宏 / Click "Stop Recording" to save

播放宏 / Playback

  1. 从下拉列表选择已录制的宏 / Select a recorded macro from dropdown
  2. 设置播放参数（循环次数、速度倍率）/ Set playback parameters
  3. 点击"播放"开始执行 / Click "Play" to start
💡 提示 / Tip: 可以勾选"录制接收数据"来同时记录设备响应，用于调试验证。
Check "Record Receive Data" to also record device responses for debugging.

2. 文件传输 File Transfer

中文：支持 XMODEM/YMODEM 协议的文件传输功能。

English: File transfer supporting XMODEM/YMODEM protocols.

支持的协议 / Supported Protocols

协议 / Protocol 数据块大小 / Block Size 特点 / Features
XMODEM 128 字节 基础校验和 / Basic checksum
XMODEM-CRC 128 字节 CRC-16校验 / CRC-16 checksum
XMODEM-1K 1024 字节 大数据块传输 / Large block transfer
YMODEM 1024 字节 支持文件名和批量传输 / Filename and batch support

发送文件 / Send File

  1. 打开 工具 → 文件传输 / Open Tools → File Transfer
  2. 选择"发送文件"模式 / Select "Send File" mode
  3. 选择传输协议 / Select protocol
  4. 点击"浏览"选择文件 / Click "Browse" to select file
  5. 点击"开始传输"/ Click "Start Transfer"

接收文件 / Receive File

  1. 选择"接收文件"模式 / Select "Receive File" mode
  2. 选择保存路径 / Select save location
  3. 点击"开始传输"等待数据 / Click "Start" and wait

3. IAP固件升级 IAP Firmware Upgrade

中文：通过串口进行在线固件升级。

English: Online firmware upgrade via serial port.

操作步骤 / Steps

  1. 确保设备处于Bootloader模式 / Ensure device is in Bootloader mode
  2. 打开 工具 → IAP升级 / Open Tools → IAP Upgrade
  3. 选择固件文件（.hex 或 .bin）/ Select firmware file
  4. 点击"开始传输"/ Click "Start Transfer"
  5. 等待传输完成 / Wait for completion
⚠️ 警告 / Warning: 固件升级过程中请勿断开连接或关闭电源！
Do not disconnect or power off during firmware upgrade!

4. 多端口管理 Multi-Port Management

中文：同时管理和监控多个串口连接。

English: Manage and monitor multiple serial port connections.

功能特点 / Features

  • 同时打开多个串口 / Open multiple ports simultaneously
  • 独立配置每个端口 / Configure each port independently
  • 支持向所有端口广播数据 / Broadcast data to all ports
  • 统计每个端口的收发字节数 / Track bytes sent/received per port

使用方法 / Usage

  1. 打开 工具 → 多端口管理 / Open Tools → Multi-Port
  2. 点击"添加端口"创建新连接 / Click "Add Port" to create connection
  3. 配置端口参数并打开 / Configure and open port
  4. 使用"发送到全部"同时发送数据 / Use "Send to All" for broadcast

功能码 / Code 名称 / Name 描述 / Description

0x06 写单寄存器 / Write Single Register 写入单个寄存器
0x0F 写多线圈 / Write Multiple Coils 写入多个线圈
0x10 写多寄存器 / Write Multiple Registers 写入多个寄存器

  • 请求/响应配对分析 / Request/Response pairing

7. 数据搜索 Data Search

中文：在接收数据中快速搜索特定内容。

English: Quickly search for specific content in received data.

使用方法 / Usage

  1. 按 Ctrl+F 或 编辑 → 数据搜索 / Press Ctrl+F or Edit → Data Search
  2. 在过滤标签页输入搜索关键词 / Enter keywords in Filter tab
  3. 匹配的行会实时显示 / Matching lines shown in real-time
💡 提示 / Tip: 搜索支持正则表达式，如 ERROR|FAIL 可同时搜索两个关键词。
Search supports regex, e.g., ERROR|FAIL searches both keywords.

  • 暂停更新 / Pause: 暂停实时更新以查看历史数据

9. 交互式控件面板 Interactive Control Panel

中文：提供可视化控件进行实时调参和数据显示。

English: Visual controls for real-time parameter tuning and data display.

打开方式 / How to Open

菜单 工具 → 控件面板 / Menu Tools → Control Panel

支持的控件 / Supported Controls

控件 / Control 功能 / Function
滑动条 / Slider 拖动调节参数，实时发送到设备
3D姿态 / 3D Attitude 显示 IMU/飞控的 Roll/Pitch/Yaw 姿态

滑动条控件 / Slider Control

功能特点 / Features:

  • 拖动滑动条实时发送数据
  • 点击名称可直接编辑
  • 自定义数值范围、步进、小数位数
  • 自定义发送模板（如 SET %s %.2f\n）
  • 发送频率限制，防止数据拥堵

右键配置 / Right-click Config: 右键点击控件 → 配置...

3D姿态控件 / 3D Attitude Control

功能特点 / Features:

  • 3D飞行器模型实时旋转显示
  • 支持 Roll/Pitch/Yaw 三轴显示
  • 鼠标拖动旋转视角，滚轮缩放
  • 显示坐标轴和参考网格
  • 可绑定数据通道自动更新

数据通道绑定 / Channel Binding:

  • 横滚通道 (Roll): 绑定接收 Roll 数据
  • 俯仰通道 (Pitch): 绑定接收 Pitch 数据
  • 偏航通道 (Yaw): 绑定接收 Yaw 数据

控件管理 / Control Management

  • 添加控件: 选择控件类型 → 点击"添加"
  • 删除控件: 右键点击控件 → 删除
  • 清空所有: 点击"清空"按钮
  • 保存配置: 点击"保存"导出为JSON文件
  • 加载配置: 点击"加载"导入JSON文件
💡 应用场景 / Applications:
PID参数在线调节、电机速度控制、飞控姿态监控、传感器数据可视化
