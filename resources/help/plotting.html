绘图功能 Plotting Features

中文：串口调试助手提供强大的实时数据绘图功能，支持多窗口、多曲线绘制。

English: ComAssistant provides powerful real-time data plotting with multi-window and multi-curve support.

1. 创建绘图窗口 / Creating Plot Windows

方法一：通过协议自动创建 / Auto-create via Protocol

中文：发送包含窗口ID的绘图数据时，系统会自动创建对应窗口。

English: Sending plot data with window ID will automatically create the corresponding window.

示例 / Example:

{plot1}1.23,4.56,7.89

将自动创建名为 "plot1" 的绘图窗口 / Automatically creates plot window named "plot1"

方法二：手动创建 / Manual Creation

中文：通过菜单 绘图 → 新建绘图窗口 手动创建。

English: Via menu Plot → New Plot Window to create manually.

2. 窗口控制 / Window Controls

功能 / Feature 说明 / Description
暂停/继续 / Pause/Resume 暂停数据更新，方便观察细节 / Pause data update for detailed observation
清空数据 / Clear Data 清除当前窗口所有数据 / Clear all data in current window
自动缩放 / Auto Scale 自动调整坐标轴范围以适应数据 / Auto adjust axis range to fit data
网格显示 / Show Grid 显示或隐藏坐标网格 / Show or hide coordinate grid
图例显示 / Show Legend 显示或隐藏曲线图例 / Show or hide curve legend
缩放/拖拽 / Zoom/Drag 鼠标滚轮缩放，左键拖拽移动 / Mouse wheel zoom, left-click drag to move

3. 曲线管理 / Curve Management

3.1 自动曲线分配 / Auto Curve Assignment

中文：每个数据字段自动分配一条曲线，颜色自动区分。

English: Each data field is automatically assigned a curve with distinct color.

示例 / Example:

{plot1}1.23,4.56,7.89
  • 曲线1（蓝色）：1.23 / Curve 1 (blue): 1.23
  • 曲线2（红色）：4.56 / Curve 2 (red): 4.56
  • 曲线3（绿色）：7.89 / Curve 3 (green): 7.89

3.2 曲线命名 / Curve Naming

中文：曲线默认命名为 "曲线1"、"曲线2"...，可在设置中自定义名称。

English: Curves are named "Curve 1", "Curve 2"... by default, customizable in settings.

4. 坐标轴设置 / Axis Settings

4.1 手动设置范围 / Manual Range Setting

中文：右键点击坐标轴，选择"设置范围"可手动指定最小值和最大值。

English: Right-click on axis, select "Set Range" to manually specify min and max values.

4.2 自动缩放模式 / Auto Scale Mode

  • 全局自动缩放 / Global Auto Scale: 根据所有数据自动调整
  • 滚动窗口 / Rolling Window: 仅显示最近N个数据点

5. 数据导出 / Data Export

格式 / Format 说明 / Description
PNG图片 / PNG Image 导出当前绘图为高清PNG图片 / Export current plot as high-resolution PNG image
JPG图片 / JPG Image 导出为JPG格式（文件更小）/ Export as JPG format (smaller file size)
CSV数据 / CSV Data 导出原始数据为CSV文件 / Export raw data as CSV file
PDF文档 / PDF Document 导出为矢量PDF文档 / Export as vector PDF document

6. 性能优化 / Performance Optimization

6.1 数据点限制 / Data Point Limit

中文：单窗口默认最多保存10,000个数据点，超出后自动删除最早的数据。

English: Single window saves max 10,000 data points by default, oldest data auto-deleted when exceeded.

💡 提示 / Tip: 可在设置中调整最大点数（1,000 ~ 100,000）
Can adjust max points in settings (1,000 ~ 100,000)

6.2 OpenGL加速 / OpenGL Acceleration

中文：启用OpenGL加速可显著提升绘图性能（推荐用于高速数据）。

English: Enable OpenGL acceleration for significant performance boost (recommended for high-speed data).

启用方法 / Enable Method:

菜单: 绘图 → 绘图设置 → 启用OpenGL加速

Menu: Plot → Plot Settings → Enable OpenGL Acceleration

6.3 数据抽稀 / Data Decimation

中文：当采样率过高时（>100Hz），建议启用数据抽稀以降低CPU占用。

English: When sampling rate is too high (>100Hz), enable data decimation to reduce CPU usage.

抽稀比例 / Decimation Ratio 说明 / Description
1:1（无抽稀）/ 1:1 (No decimation) 保留所有数据 / Keep all data
1:2 每2个点保留1个 / Keep 1 out of every 2 points
1:5 每5个点保留1个 / Keep 1 out of every 5 points
1:10 每10个点保留1个 / Keep 1 out of every 10 points

7. 高级功能 / Advanced Features

7.1 多窗口同步 / Multi-Window Sync

中文：启用后，多个绘图窗口可以同步操作。

English: When enabled, multiple plot windows can be synchronized.

启用方法 / How to Enable: 菜单 绘图 → 窗口同步

同步内容 / Sync Features:

  • 暂停/继续状态 / Pause/Resume: 一个窗口暂停，所有窗口同步暂停
  • X轴范围 / X-Axis Range: 拖动缩放时，其他窗口同步调整
  • 清空操作 / Clear: 同步清空所有窗口数据
💡 应用场景 / Applications:
对比多个通道数据、同时观察多个传感器、分析不同数据源的时间相关性

7.2 直方图视图 / Histogram View

中文：将曲线数据以直方图形式显示，用于分析数据分布特征。

English: Display curve data as histogram for analyzing data distribution.

启用方法 / How to Enable: 菜单 视图 → 直方图

配置选项 / Configuration:

  • 数据源曲线 / Source Curve: 选择要分析的曲线
  • 分箱数量 / Bin Count: 10-200，数量越多分辨率越高

显示信息 / Display Info:

  • 数据分布柱状图
  • 均值 (Mean)
  • 标准差 (Std Dev)
  • 最大/最小值

7.3 XY 绘图模式 / XY Plot Mode

中文：将两个通道的数据作为X-Y坐标绘制，适用于李萨如图形、相位分析等。

English: Plot two channels as X-Y coordinates, suitable for Lissajous figures, phase analysis, etc.

启用方法 / How to Enable: 菜单 视图 → XY模式 或快捷键 Ctrl+Y

配置选项 / Configuration:

  • X通道 / X Channel: 选择作为X轴的数据通道
  • Y通道 / Y Channel: 选择作为Y轴的数据通道

应用示例 / Examples:

  • 发送正弦和余弦信号，显示圆形（李萨如图形）
  • 分析两个传感器数据的相关性
  • 电机速度-位置相图分析
💡 提示 / Tip:
切换回时间序列模式：菜单 视图 → 时间序列

7.4 数据标记 / Data Markers

中文：鼠标悬停在曲线上可显示精确数值。

English: Hover mouse over curve to display precise values.

7.5 曲线样式 / Curve Style

中文：支持线型（实线、虚线）、点型（圆点、方块）、线宽调整。

English: Supports line style (solid, dashed), point style (circle, square), line width adjustment.

7.6 FFT 频谱分析 / FFT Spectrum Analysis

中文：对曲线数据进行快速傅里叶变换（FFT）分析，可查看频谱特性。

English: Perform Fast Fourier Transform (FFT) analysis on curve data to view spectrum characteristics.

使用方法 / How to Use:

  • 中文：菜单 分析 → FFT 频谱分析 或快捷键 Ctrl+F
  • English: Menu Analysis → FFT Spectrum Analysis or shortcut Ctrl+F

分析结果包含 / Analysis Results Include:

  • 幅度谱 / Magnitude Spectrum: 显示各频率分量的幅度
  • 相位谱 / Phase Spectrum: 显示各频率分量的相位（-180° ~ 180°）
  • 功率谱 / Power Spectrum: 显示各频率分量的功率
  • 分析结果 / Analysis Results: 主频率、总功率、DC分量等统计信息
💡 提示 / Tip:
中文： 数据点数越多，频率分辨率越高。建议至少采集100个数据点进行分析。
English: More data points provide higher frequency resolution. Recommend at least 100 data points for analysis.

7.5 测量游标 / Measurement Cursors

中文：使用两个可移动游标精确测量曲线上任意两点之间的差值。

English: Use two movable cursors to precisely measure the difference between any two points on the curve.

使用方法 / How to Use:

  • 中文：菜单 分析 → 测量游标 或快捷键 Ctrl+M
  • English: Menu Analysis → Measurement Cursors or shortcut Ctrl+M

功能说明 / Features:

  • 设置游标1和游标2的X位置
  • 自动显示两游标之间的ΔX和ΔY
  • 可用于测量周期、幅度等参数

7.6 数字滤波 / Digital Filtering

中文：对曲线数据应用数字滤波，去除噪声或提取特定频率成分。

English: Apply digital filtering to curve data to remove noise or extract specific frequency components.

支持的滤波器类型 / Supported Filter Types:

滤波器 / Filter 用途 / Usage
滑动平均 / Moving Average 平滑数据，去除高频噪声
中值滤波 / Median 去除脉冲噪声和异常值
指数移动平均 / EMA 实时平滑，响应更快
低通滤波 / Low-pass 去除高频噪声，保留低频信号
高通滤波 / High-pass 去除直流分量和低频漂移
带通滤波 / Band-pass 提取特定频段信号
带阻滤波 / Band-stop 去除特定频段噪声（如工频干扰）
卡尔曼滤波 / Kalman 自适应滤波，适合动态系统

7.7 触发捕获 / Trigger Capture

中文：类似示波器的触发功能，在满足条件时捕获数据。

English: Oscilloscope-like trigger function to capture data when conditions are met.

触发类型 / Trigger Types:

  • 上升沿 / Rising Edge: 信号从低于阈值上升到高于阈值时触发
  • 下降沿 / Falling Edge: 信号从高于阈值下降到低于阈值时触发
  • 高电平 / Level High: 信号高于阈值时触发
  • 低电平 / Level Low: 信号低于阈值时触发

配置选项 / Configuration:

  • 触发源曲线选择
  • 触发电平设置
  • 触发前/后数据点数
  • 单次触发模式

7.8 峰值自动标注 / Peak Auto-Annotation

中文：自动检测并标注曲线上的极值点（极大值和极小值）。

English: Automatically detect and annotate extreme points (maxima and minima) on curves.

配置选项 / Configuration:

  • 目标曲线选择
  • 峰值检测阈值（相对于振幅）
  • 相邻峰值最小间距
  • 显示极大值/极小值选项
  • 最大标注数量限制

8. 嵌入式工程师功能 / Embedded Engineer Features

8.1 增强统计分析 / Advanced Statistics

中文：提供嵌入式开发常用的统计指标，帮助分析信号质量。

English: Provides statistics commonly used in embedded development to help analyze signal quality.

使用方法 / How to Use: 菜单 分析 → 增强统计分析

统计指标 / Statistics:

指标 / Metric 说明 / Description
最小值/最大值 / Min/Max 数据的极值范围
峰峰值 / Peak-to-Peak 最大值与最小值之差，表示信号振幅
平均值 / Average 数据的算术平均
中位数 / Median 数据的中间值，不受异常值影响
标准差 / Std Dev 数据的离散程度
RMS / RMS 均方根值，用于表示交流信号有效值
波峰因子 / Crest Factor 峰值与RMS之比，用于评估信号波形
采样率 / Sample Rate 估算的数据采样率(Hz)
💡 提示 / Tip:
中文： 使用测量游标选择区间后，可以对选定区间进行统计分析。
English: Use measurement cursors to select a range, then perform statistics on the selected interval.

8.2 数据报警系统 / Data Alarm System

中文：实时监控数据，当超出设定范围时触发报警。

English: Real-time data monitoring with alarms when values exceed set limits.

使用方法 / How to Use: 菜单 分析 → 数据报警配置

报警类型 / Alarm Types:

  • 上限报警 / Upper Limit: 数值超过上限时报警
  • 下限报警 / Lower Limit: 数值低于下限时报警
  • 变化率报警 / Rate Alarm: 数值变化过快时报警

报警动作 / Alarm Actions:

  • 声音提示（蜂鸣）
  • 视觉提示（界面闪烁）
  • 自动暂停数据采集
⚠️ 应用场景 / Applications:
中文： 适用于电压监测、温度监控、传感器数据异常检测等场景。
English: Suitable for voltage monitoring, temperature monitoring, sensor anomaly detection, etc.

8.3 数据标记功能 / Data Markers

中文：在曲线上添加自定义标记和注释，方便标注关键数据点。

English: Add custom markers and annotations on curves to highlight key data points.

使用方法 / How to Use: 菜单 分析 → 添加数据标记

标记功能 / Features:

  • 选择目标曲线
  • 指定X/Y位置（可自动从曲线获取Y值）
  • 自定义标记文本
  • 自定义标记颜色
  • 批量清除所有标记

8.4 嵌入式场景预设 / Embedded Scene Presets

中文：针对常见嵌入式应用场景的一键预设配置。

English: One-click preset configurations for common embedded application scenarios.

使用方法 / How to Use: 菜单 分析 → 嵌入式场景预设

预设场景 / Preset Scenes:

场景 / Scene 配置说明 / Configuration
PID 调参 / PID Tuning 设定值、实际值、误差、输出 四条曲线
适合PID参数整定和响应分析
电机控制 / Motor Control 速度、位置（右Y轴）、电流、目标速度
位置曲线使用独立Y轴
传感器监测 / Sensor Monitor 4通道传感器数据同时显示
不同颜色区分各通道
ADC 采样 / ADC Sampling ADC原始值、滤波值、参考电压
Y轴预设0-4095（12位ADC）
电源监测 / Power Monitor 电压、电流（右Y轴）、功率
预设电压上下限报警
温度控制 / Temperature Control 目标温度、实际温度、加热功率
预设温度上限报警
💡 提示 / Tip:
中文： 选择场景后，按预设的曲线顺序发送数据即可。例如PID场景按"设定值,实际值,误差,输出"顺序发送。
English: After selecting a scene, send data in the preset curve order. E.g., PID scene: "setpoint,actual,error,output".

9. 使用建议 / Usage Recommendations

场景 / Scenario 建议设置 / Recommended Settings
低速数据（<10Hz）/ Low-speed data 无抽稀，显示所有点 / No decimation, show all points
中速数据（10-100Hz）/ Medium-speed data 1:2抽稀，保留5000点 / 1:2 decimation, keep 5000 points
高速数据（>100Hz）/ High-speed data 1:5抽稀，启用OpenGL / 1:5 decimation, enable OpenGL
多曲线（>8条）/ Multi-curve (>8) 降低线宽，启用OpenGL / Reduce line width, enable OpenGL
⚠️ 注意 / Warning:
中文： 单窗口曲线数量建议不超过16条，过多曲线会影响性能和可读性。
English: Recommend ≤16 curves per window, too many curves affect performance and readability.
